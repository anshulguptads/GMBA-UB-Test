# 🏦 Universal Bank Analytics Dashboard

A comprehensive Streamlit dashboard for analyzing Universal Bank's personal loan campaign data.

## 📊 Features

### 1. Descriptive Analytics
- Summary statistics
- Distribution plots
- Categorical analysis
- Cross-tabulations

### 2. Diagnostic Analytics
- Correlation heatmap
- Segment comparison
- Feature relationships

### 3. Predictive Analytics
- 7 ML models comparison
- ROC curves
- Feature importance
- Interactive prediction tool

### 4. Prescriptive Analytics
- Customer scoring
- Segment strategy
- Campaign ROI calculator
- Strategic recommendations
